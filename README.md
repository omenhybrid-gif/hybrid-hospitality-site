# Hybrid Hospitality Site

## Run locally

```bash
npm install
npm run dev
```

## Deploy to Vercel

1. Upload this folder to GitHub.
2. Go to Vercel and import the GitHub repository.
3. Click Deploy.

## Images

This starter includes simple placeholder image files in `/public` so the site can load immediately.
Replace these files later with your real airport photos:
- `public/airport-2.jpg`
- `public/airport-3.jpg`
- `public/airport-4.jpg`
